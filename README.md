# US-Healthcare-Analytics-Case-Study-Excel-
Project Overview

This project analyzes healthcare data from 100 hospitals using Microsoft Excel to identify patterns in patient demographics, medical conditions, healthcare costs, and hospital resource utilization.

Objectives
Analyze medical conditions across demographic groups.
Identify healthcare cost drivers and optimization opportunities.
Study admission trends for better hospital resource planning.
Create executive-level dashboards and reports.
Excel Features Used
Pivot Tables
Pivot Charts
Power Query
Data Cleaning
Conditional Formatting
Lookup Functions (XLOOKUP/VLOOKUP)
Dashboard Design
Data Analysis
Key Insights
Identified the most common medical conditions among different age groups.
Discovered factors contributing to higher patient billing amounts.
Analyzed admission trends to improve resource allocation and workload management.
Created interactive dashboards for executive reporting.
